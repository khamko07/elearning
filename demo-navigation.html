Navigation System Demo

Đây là demo cho hệ thống navigation mới với các tính năng:

  • ✅ Responsive header navigation với logo và user menu
  • ✅ Mobile hamburger menu với smooth animations
  • ✅ Search functionality trong header
  • ✅ Notification dropdown với badge
  • ✅ User avatar dropdown menu
  • ✅ Theme toggle button
  • ✅ Breadcrumb navigation system
  • ✅ Touch-friendly navigation elements
  • ✅ Keyboard navigation support

Hướng dẫn sử dụng

Desktop

  • Click vào các navigation links để điều hướng
  • Sử dụng search box để tìm kiếm
  • Click vào notification bell để xem thông báo
  • Click vào avatar để mở user menu
  • Sử dụng Ctrl+K để focus vào search

Mobile

  • Click vào hamburger menu (☰) để mở mobile menu
  • Swipe hoặc click overlay để đóng menu
  • Tất cả chức năng đều touch-friendly
  • Navigation tự động đóng khi chọn link
